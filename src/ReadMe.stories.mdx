# Welcome to YATODO

**YATODO** (yet another todo) is a single page web application developed to address the
[**Hasura recruitment exercise**](https://hasura.io/careers/). It is a simple todo list management application. It is
based on the provided  **hiring-nicolas-2** graphql schema.

# Features

### Business features:

The todo list is composed by projects regrouping tasks. Each project has a name and color and can be **active** or **archived**.
Each task has a content and a long description and can be **undone** or **done**.

- **Dashboard**, with recent active projects and pending tasks, tasks can be marked as done,
- **Projects list**, with all active and archived projects,
- **Tasks list**, with all pending and done tasks, tasks can be marked as done or undone,
- **Project page**, with all tasks of the project, with a button to archive or unarchive a project,
- **Project creation**, with random project color generation,
- **Task creation**.

### Design features:

It is possible to switch between the **light** and **dark** theme.

**Custom logo** and **loader** have been designed and implemented.

**Accessibility** has been taken into account (see Storybook a11y addon for testing, some colors are not accessible
but have been left as is).
